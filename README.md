This is an react app for the Warmpaws-Pet Care Center.
This project has been completely made with React.js.
Key features:


    1. It is using Firebase authentication system from Google.

    2. It has a regular Home page where services and some informations are shared of the pet shop their characteristics, features, services their vets and some tips for the pet care on the Home page.

    3. On the Services page it is showed which services they actually provide with a View details button.

    4. Clicking on the View Details Button will take the user to it's details page which is a private route. User must sign it or register to use the services.

    5. It has a profile page for the user where it will the show the users informations with that and profile updating page. Where User can update their name and image.

    6. Once the user logs in it will show a Avatar in white on the right top on the page just after the Log In Button. If the user is Logged in the button will change to Sign Out button.

    7. Clicking on the avatar will also redirect the user to the Profile page. Hovering it will show the users name.

    8. It's theme is based on the winter so I have made it frosty with blurring effect theme.

    1. AOS Animation.
